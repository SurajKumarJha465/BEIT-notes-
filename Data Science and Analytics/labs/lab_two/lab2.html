Theory¶

NumPy¶

NumPy is a Python library used for numerical and scientific computations. It provides support for large multidimensional arrays and matrices along with various mathematical functions to operate on them efficiently.

Features of NumPy¶

  • Fast array processing
  • Multidimensional array support
  • Mathematical and statistical functions
  • Memory efficient operations

Pandas¶

Pandas is a Python library used for data analysis and manipulation. It provides two major data structures:

  • Series
  • DataFrame

Features of Pandas¶

  • Easy data handling
  • Data cleaning and filtering
  • CSV file handling
  • Statistical analysis tools

Matplotlib¶

Matplotlib is a visualization library in Python used to create graphs and plots.

Types of Graphs¶

  • Line Plot
  • Scatter Plot
  • Bar Graph
  • Pie Chart

Software Requirements¶

5. Statistical Calculations¶

In [5]:
from statistics import mode
import numpy as np

data = np.array([10, 20, 30, 40, 50])

mean = np.mean(data)
median = np.median(data)
mode_value = mode(data)
std = np.std(data)
variance = np.var(data)

print("Mean:", mean)
print("Median:", median)
print("Mode:", mode_value)
print("Standard Deviation:", std)
print("Variance:", variance)

actual = np.array([10, 20, 30])
predicted = np.array([12, 18, 33])

mse = np.mean((actual - predicted) ** 2)
rmse = np.sqrt(mse)

print("MSE:", mse)
print("RMSE:", rmse)

9. Data Visualization Using Matplotlib¶

In [9]:
import matplotlib.pyplot as plt

subjects = [
    "Probability and Statistics",
    "Data Science and ML",
    "Artificial Intelligence",
    "Computer Graphics",
    "Engineering Economics",
    "Research Fundamentals"
]

marks = [60, 45, 59, 72, 80, 91]

# Line Plot
plt.plot(subjects, marks)
plt.xlabel("Subjects")
plt.ylabel("Marks")
plt.title("Line Plot")
plt.xticks(rotation=15)
plt.show()

# Scatter Plot
plt.scatter(subjects, marks)
plt.xlabel("Subjects")
plt.ylabel("Marks")
plt.title("Scatter Plot")
plt.xticks(rotation=15)
plt.show()

# Bar Graph
plt.bar(subjects, marks)
plt.xlabel("Subjects")
plt.ylabel("Marks")
plt.title("Bar Graph")
plt.xticks(rotation=15)
plt.show()

# Pie Chart
plt.pie(marks, labels=subjects, autopct='%1.1f%%')
plt.title("Pie Chart")
plt.show()
